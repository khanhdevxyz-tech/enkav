## ⚡ Core Features
*   **Deep Encryption**: Sử dụng cơ chế nén kép **LZMA** và **Zlib** để bao bọc mã nguồn.
*   **Han-Character Variable Mapping**: Thay thế logic nội bộ bằng ma trận **Hán tự** (`魂`, `魔`, `死`...) để phá vỡ các trình dịch ngược.
*   **Anti-Debug Shield**: Tích hợp các lớp bảo vệ phát hiện debugger như `gettrace` hoặc `_getframe`.
*   **Signature Branding**: Tự động đóng dấu bản quyền `𝐨𝐛𝐟 𝐛𝐲 𝐞𝐤𝐚𝐯 𝐝𝐢𝐧𝐡 𝐯𝐚𝐧 𝐤𝐡𝐚𝐧𝐡` vào file kết quả.

## 🚀 Quick Start
1.  **Clone**: 
    ```bash
    git clone [https://github.com/khanhdev/ekav-obf.git](https://github.com/khanhdev/ekav-obf.git)
    ```
2.  **Run**: 
    ```bash
    python main.py
    ```
3.  **Input**: Nhập tên file `.py` cần "phong ấn".
4.  **Result**: Nhận file đã bảo vệ tại `obf-<filename>.py`.
